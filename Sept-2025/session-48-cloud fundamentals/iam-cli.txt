https://docs.aws.amazon.com/cli/latest/reference/iam/

to list all users: aws iam list-users

get individual user details:
    aws iam get-user \
    --user-name SonamSoni

attach user policy:
    aws iam attach-user-policy \
    --policy-arn arn:aws:iam::aws:policy/AdministratorAccess \
    --user-name SonamSoni
